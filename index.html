Projects

Reddit Minimal

This project involved designing minimal version of Reddit to provide users with a streamlined and visually appealing Reddit experience. The project is built using HTML, CSS, JavaScript, React, and Redux.

Full stack Application

In this project, I developed a full-stack E-Commerce Application, encompassing features like user account creation, authentication, product addition to the cart, order history viewing, payment processing, third-party service authentication and many more. The application was constructed utilizing PostgreSQL, Node.js, Express.js, React.js, and Redux.

RSS Feed Filter

This project involved developing a Python program using object-oriented programming principles to filter RSS feeds. The program is designed to read in RSS feeds and filter out unwanted information based on specific criteria.

Spotify Playlist Builder

In this project, i build a React web application called Jammming. I used my knowledge of React components, passing state, and requests with the Spotify API to build a website that allows users to search the Spotify library, create a custom playlist, then save it to their Spotify account.

Recursion and Encryption

This project involved developing a Python program to encrypt messages using the Caesar Cipher method. The program employs concepts such as recursion and object-oriented programming to achieve the encryption process.

Word Game

This project involved developing a word game in Python using a dictionary. The game presents users with a series of letters and challenges them to create as many valid words as possible using those letters.

Computer from Scratch

In my MIT 6.004 computer architecture course, I took on the challenge of creating a CPU from basic principles. Starting with the basics of bit encoding, I worked through MOSFETs, combinational and sequential logic, and finite state machines. I pieced together a CPU step by step, turning abstract concepts into a functional unit. This hands-on project not only improved my technical skills but also deepened my understanding of computing systems. From binary basics to a custom CPU, it's been an exciting journey of learning and creating.
